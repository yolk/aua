Aua
=============

aua = a user agent (parser).

Installation
-------

    gem install aua


Basic Usage
-------

    Aua.parse("Mozilla/5.0 (Macintosh; U; Intel Mac OS X 10.6; de; rv:1.9.2.13) Gecko/20101203 Firefox/3.6.13")
